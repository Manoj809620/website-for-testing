## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* npm
  ```bash
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/your_username/repo_name.git
   ```
2. Install NPM packages
   ```bash
   npm install
   ```
3. Run the development server
   ```bash
   npm run dev
   ```
